# 114-2_EmbeddedSystemWorkshop

📘 課程亮點：四步驟成就你的 AIoT 專案

### Step 1：核心建置 ([Lab1](./Lab1.md))
快速上手 Raspberry Pi 4.0，從系統燒錄到環境優化，打好穩固基礎。

### Step 2：視覺智慧 (([Lab2]([Lab9 - Install OpenCV and Deep Learning](/114-2_EmbeddedSystemWorkshop/assets/Lab9%20Install%20opencv%20and%20Deep%20learning_v1.pdf))
實作 OpenCV 與深度學習框架，讓你的樹莓派「看懂」世界，掌握 AI 影像辨識核心。

### Step 3：無線鏈結 (Lab 3)
深入 WiFi 與 Bluetooth 設定，學習如何讓裝置與外界無縫溝通，實現真正的聯網功能。

### Step 4：雲端管控 ([Lab4](./Lab4.md))
部署 Code Server 遠端開發環境，體驗隨時隨地讀取資料、修改程式與操控裝置的便利。
